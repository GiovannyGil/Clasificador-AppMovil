# Clasificador-AppMovil
Sistema de clasificación de imágenes y una app WEB/MOVIL con Ionic

en este proyecto trabaje ayudando a crear un modelo de clasificación de imagenes, pertenecientes a categorias de maderas, de la cual se logró.
posteriormente se solicitó que se trabaja en una nueva version del modelo pero con la tecnologia perteneciente a TensorFlow.js de "Augment Data"
para crear más datos de prueba, en este caso imagenes, basada en las que ya se tenia, para que el modelo fuera más preciso

despues de un par de semanas trabajando en la nueva versión se me pidió que dejera como estaba, que ya no se iba a trabajar más en este proyecto


se me pidio que adicional al modelo se me pidio una app movil con el Framework Ionic, es la cual se pudiera subir una images y clasificarla
tambien con la opcion de tomar una foto y clasificarlar, en una primera version se logro terminar la app con los primeros requisitos ya mencionados.
